Usage

Get ipa

  1. Install EagleGamer
  2. Connection BlueTooth
  3. Open EagleGamer.
    • Select Category TabBar.
    • Select One Category Emulator.
    • Download Game
    • Select My Games TabBar.
    • Open Downloaded Game.
    • Click Resume on the left top button.

  4. GamePad tap Home and .(one dot) in KeyBoard Mode.
  5. Now, you can play. Just Go!
